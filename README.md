# React movies page
React SPA that uses an [public movie API](https://yts.mx/api) and connect with express [server](https://github.com/carlosEdua/movie-server) 

## Available Scripts

In the project directory, you can run:

### `yarn start`

### `yarn test`

### `yarn build`
